# Changelog

All notable changes to AyoChip will be documented in this file.

The format is based on Keep a Changelog.

---

# [1.0.0] - 2026-05-16

## Added

### Framework
- Added AyoChip core framework
- Added modular package structure
- Added Rojo project support
- Added Wally package manager support
- Added Selene configuration
- Added StyLua formatting configuration

### UI
- Added CustomFrame UI system
- Added rounded UI support
- Added UIStroke support
- Added dark neon theme
- Added reusable UI module system

### Systems
- Added Inventory system
- Added Notification system
- Added ToolFramework system
- Added ThemeManager module
- Added lightweight Signal system
- Added Promise package support
- Added Maid cleanup utility

### Server
- Added PlayerAdded handling
- Added starter server initialization
- Added framework boot logging

### Client
- Added ClientInit loader
- Added starter client initialization

### Project
- Added README documentation
- Added project folder structure
- Added RBXM export support
- Added GitHub release structure
- Added Windows/Linux/macOS build layout
